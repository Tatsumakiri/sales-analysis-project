# Sales Analysis Project

Python を使用して販売データの分析を行ったデータ分析学習プロジェクトです。

## Overview

本プロジェクトは、Python を利用して販売データの分析を行う学習用プロジェクトです。

顧客ごとの購入傾向や商品別の販売状況を分析し、データの集計・可視化を行いました。

データ分析の基礎であるデータ前処理、集計、可視化、分析結果の確認を学習することを目的として作成しました。

また、Jupyter Notebook を利用して分析過程を確認できるように構成しています。

## Run Project

### 仮想環境作成

```bash
python -m venv venv
```

### 仮想環境有効化

#### Mac / Linux

```bash
source venv/bin/activate
```

#### Windows

```bash
venv\Scripts\activate
```

### ライブラリインストール

```bash
pip install -r requirements.txt
```

### Jupyter Notebook 起動

```bash
jupyter notebook
```
