# 販売データ分析プロジェクト

## 概要
本プロジェクトは、Pythonを使用して販売データを分析する学習用プロジェクトです。
顧客ごとの購入傾向や商品の販売状況を確認し、データの集計・可視化を行いました。

データ分析の基本であるデータ読み込み、前処理、集計、可視化の流れを理解することを目的としています。

## 使用技術
- Python
- pandas
- matplotlib
- Jupyter Notebook
- Git / GitHub

## 分析内容
- 販売データの読み込み
- 欠損値やデータ形式の確認
- 顧客ごとの購入傾向分析
- 商品別の販売数集計
- グラフによる可視化

## 工夫した点
単にデータを表示するだけでなく、顧客ごとの購入パターンや商品ごとの販売傾向が分かるように集計しました。
また、分析結果をグラフで可視化することで、数値だけでは分かりにくい傾向を確認できるようにしました。
